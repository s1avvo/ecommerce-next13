# Simply e-commerce application.
This is a project made to complete course NEXT 13 MASTERS from BRAVE COURSES SP. Z O.O.

https://ecommerce-next13-nine.vercel.app/

## General Info
A comprehensive and efficient e-commerce application built in the Next 13 App Router, with the implementation of the Strapi payment module and Clerk user registration.

## Features

- Hygraph API Integration.
- Active Link navigation through categories and products for a better user experience.
- Pagination.
- Search bar implementation for quick and efficient product searches.
- Dynamic SEO Metadata for enhanced product visibility.
- Open Graph Images for each single product.
- Product reviews with optimistic updates.
- Sorting by price and average rating.
- Suggested products with Typesense vector search - utilize advanced vector search for personalized product suggestions.
- Shopping cart functionality.
- Strapi payment module.
- Cler authentication module.
